# Test Outputs

Esta carpeta guarda los outputs reales de prueba del nuevo sistema editorial de Triggui.

Aquí no van prompts.  
Aquí no van fixtures.  
Aquí no van rúbricas.

Aquí van únicamente:

- respuestas generadas por modelos
- evaluaciones de esas respuestas
- comparaciones entre sistemas o variantes
